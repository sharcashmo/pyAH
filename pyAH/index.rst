==================
pyAH documentation
==================

These are the documentation pages of pyAH project, and Atlantis Helper developed
in `Python <http://www.python.org>`_ and `wxPython <http://www.wxpython.org>`_.

Project has been developed to provide general tools to work with Atlantis. So
:obj:`!atlantis.parsers` sub-package implements generic parsing tools that can
be used by other projects others Atlantis PBEM players can develop, not only
for :ref:`pyAH <pyAH-project>`.
