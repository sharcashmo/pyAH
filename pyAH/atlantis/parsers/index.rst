.. _atlantis.parsers:

~~~~~~~~~~~~~~~~~~~~~~~
:obj:`atlantis.parsers`
~~~~~~~~~~~~~~~~~~~~~~~

:ref:`atlantis.parsers` subpackage contains some modules devoted to Atlantis
PBEM text files parsing. Up to the date only two Atlantis PBEM files are parsed:
report file and orders file.

Each module in :ref:`atlantis.parsers` defines classes needed to parse one of
these files. The structure is similar for all of them: there's a *parser* class
which is in charge of parsing input lines and calls its *consumer* with the
parsed data; and a *consumer* interface called by the *parser* which needs to
be implemented by any class willing to consume data generated by the *parser*.

In addition some helper classes may be defined, also. For example
:mod:`atlantis.parsers.reportparser` defines a
:class:`~atlantis.parsers.reportparser.ReportReader` class to unwrap report
lines before being parsed by
:class:`~atlantis.parsers.reportparser.ReportParser`.
